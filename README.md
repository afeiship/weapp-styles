# weapp-styles
> Weapp styles all in one.

[![license][license-image]][license-url]
[![size][size-image]][size-url]

## installation
```shell
npm i @jswork/weapp-styles
```

## usage
```scss
// use sass
@import '~@jswork/weapp-styles/dist/index.scss';
// use css
@import '~@jswork/weapp-styles/dist/style.css';
```
